## 项目运行

首先保证自己机器 Node.js 版本在 18 及以上！！！

1. 拉取项目

```
git pull https://gitee.com/powerRock/sugarcode-blog.git
```

2. 安装 vitepress

```
npm install vitepress
```

3. 启动服务

```
npm run docs:dev

npx vitepress dev docs
```

4. 项目打包

```
npm run docs:build
```
